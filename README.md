# Modern Calculator

A sleek and modern calculator built with Python and Tkinter.  
Supports basic arithmetic operations (+, -, *, /), parentheses, and decimals.  
Dark-themed interface with clickable buttons and a clear (C) function.

## Features
- Modern dark interface
- Buttons for numbers and operators
- Clear (C) button to reset the display
- Works as Python script (`calculator.py`) or Windows executable (`calculator.exe`)

## How to Use
1. Run `calculator.py` with Python 3 to use the script.
2. Or download `calculator.exe` to run it directly on Windows.
